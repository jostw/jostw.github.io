2015 面試總結

01 November 2015 on 2015, 前端, 面試, JavaScript

今年五月底離職，準備換工作。待業了將近五個月，算是跟預期的時程差不多，不過預期之外的是最後去了一間本來不在口袋名單內的公司。

六月跟大學同學去了關西玩了將近十天，剩下的日子算是在放鬆休息，認真練習吉他，然後開始規律的運動。

七月很認真的把一些之前想寫想玩的東西摸一摸，大部分的成果都放在 GitHub 上，不過最後還是有些東西來不及摸 XD

八月開始準備個人網站以及履歷，九月開始面試，然後一邊修網站及履歷。

十月中收到 offer，預計十一月 on board。

這次面試只準備了英文履歷，然後除了主動投的之外，只有透過 LinkedIn 還有 head hunter 去接洽。真正有面試到的公司並沒有很多，其實是因為口袋名單內的公司都沒有回應，並且很順利的就拿到一個還不錯的 offer，所以最後就決定接受不繼續尋找了。

這次找工作的心得是，比起第一次找工作來說，有工作經驗之後真的比較好找。待業時間有點長，雖然是在規劃之中，但是到了七八月其實開始有點慌，除了會被家人朋友問之外，自己也會感覺是不是要修改一下計畫，加速整個時程。不過最終還是照著原先的規劃走，雖然默默想了很多後續的 BCDEFG 計畫 XD

面試過程中幾乎都是被問到 JavaScript 的問題，最常見不外乎 closure, scope, bind 等等。如果對 JavaScript 觀念不熟的話，這些問題一問就知道程度在哪裡了。當然對我來說比較苦手的就是如何用文字去描述這些東西，尤其是用英文。寫的時候知道要怎麼寫，但要解釋還真的不容易，我想這也是未來要加強的地方。

最終接受的 offer 其實本來並沒有在口袋名單中，不過在面試過程中有被打動，並且在各方面來說其實都有符合期待，所以最後就接受了。

大致上就是這次面試過程的感想，敬請期待下次的面試文(?)。
